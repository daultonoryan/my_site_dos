A Django Powered Resume

Hello! This is my personal site and resume, a (near) exhaustive rundown of all the projects I have worked on and all the interests that have captivated my attention at one time or another. Studying biology in college, I realized eventually that my real passion lay in computer science.

I graduated from VCU in 2016, and not long after that, I set out to teach myself how to code. Since then, I have amassed a strong base of knowledge, having mastered multiple core concepts and languages. The language that I get most excited about? That’s easy: Python! This language is highly readable, infinitely flexible, and quickly adapted to a huge number of applications.

For this site specifically, I used HTML and CSS on the frontend and Python’s Django library on the backend. Designed to take advantage of most of the stunning features Django has to offer, I have taken the construction of this site as an opportunity to practice and display good coding practices. If you are already falling asleep because you view coding as about as exciting as watching grass grow, fear not, I won’t bore you any further. If however, you view coding as an exciting and creative topic where you get to create something beautiful from nothing then I fully invite you to check out my source code which can be found here.

I’m happy to share my resume with you. If there are any questions I can answer for you or if you are interested in freelance work or hiring for a permanent position, please feel free to reach out to me!
